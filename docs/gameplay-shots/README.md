# 玩法连拍素材集

这里是自由探索各玩法的 3D 截图素材，由离线截图脚本 `frontend/scripts/_xy_play.mjs`
（Playwright + SwiftShader 软件 WebGL，`?perf=high` 满配渲染）连拍产出，多为同一场景的
多角度 / 连续帧候选。主 [README](../../README.md) 的「界面预览」已从中精选代表图入库
（`docs/screenshots/12~15`），此目录保留完整素材以备复用。

| 分组 | 文件 | 说明 |
|---|---|---|
| 暮色天灯（第二批） | `lanternC_0` ~ `lanternC_5` | 夜村放飞天灯的 6 连拍，含烟花绽放与天灯升空；质量最佳的一批 |
| 暮色天灯（第一批） | `lantern_0` ~ `lantern_4` | 早期天灯连拍 5 张 |
| 开车兜风 | `drive_0` ~ `drive_2` | 林间土路驾驶，黄色跑车 / 彩虹 / 夕照（`drive_2` 已入主 README） |
| 拾海垂钓 | `fish_0` ~ `fish_3`、`fish_near` / `fish_over` / `fish_side` | 海湾抛竿的连续帧与近 / 俯 / 侧多机位（`fish_over` 已入主 README） |
| 风铃心曲 | `chime_a` / `chime_b` / `chime_c` | 夜色风铃区三个机位（`chime_a` 已入主 README） |

> 这些是展示素材，非运行时资源；前端不引用此目录。原始 PNG 在本地 `frontend/_audit_play/`
> （已被 `.gitignore` 忽略），此处为 JPEG（quality 90）压缩归档版。
